# Blog Backend (Express + MongoDB)

Quick start:

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies (already done if initialized):

```powershell
npm install
```

3. Run in development:

```powershell
npm run dev
```

Health check: GET http://localhost:4000/health

This repo contains the backend API skeleton for the Blog MVP. Next steps: add models, auth routes, posts and comments routes, and tests.
